# Examples

This directory contains example CHARACTER.md projects at different levels of complexity.

## [minimal/](minimal/)

A single-file CHARACTER.md for a TRPG-style character named Aldric. Everything — dispositions, knowledges, and experiences — lives in one file. Good for small characters, quick prototypes, or getting a feel for the format.

## [full/](full/)

A complete CHARACTER.md project for 奎 (Kuei), the protagonist of *The Stolen Prototype* (同步戰紀：失竊的原型機) by [Narrativesaw](https://narrativesaw.com). Demonstrates the full directory structure: a main file plus dispositions/, knowledges/, and experiences/ folders.

This character was extracted from a published novel — the same process described in the README's "source text" scenario. For a live version you can talk to, see [book-mcp](https://github.com/liheng-personal/book-mcp).
